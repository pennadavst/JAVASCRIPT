* The main focus of the projects in this repository is to show the effective use of JAVASCRIPT.
* All the projects are responsive for all screen sizes.
* The breakpoints for the screen sizes are as follows:
    *Small- 576px
    *Medium- 768px
    *Large- 992px
    *Extra large- 1200px
                              -(px= Pixels)
* Vanilla JavaScript is used to develop the projects and no JavaScript framework is used.
